SCENARIO 1:E Commerce analysis
https://www.kaggle.com/datasets/carrie1/ecommerce-data
Code description:
This Python program analyzes and visualizes sales data using the Pandas and Matplotlib libraries. The dataset is loaded and inspected to understand its structure and missing values. Invalid records with zero or negative quantity and unit price are removed. A new column, TotalSales, is created to calculate the total value of each transaction. The data is grouped by product description to identify and visualize the top ten products based on total sales using a bar chart. The invoice date is converted into datetime format, and daily total sales are plotted using a line graph to study sales trends over time. Overall, the program helps analyze product performance and sales patterns effectively.
